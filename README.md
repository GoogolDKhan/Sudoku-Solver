# [Sudoku Solver](https://www.freecodecamp.org/learn/quality-assurance/quality-assurance-projects/sudoku-solver)

## Tech Stack

**Client:** HTML, CSS, JS

**Server:** Node, Express

**Testing:** Mocha, Chai

## Run Locally

```bash
git clone https://github.com/GoogolDKhan/Sudoku-Solver.git

cd Sudoku-Solver

npm install

npm run start
```
